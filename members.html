Argonaut

Position:
Leader / In Game Leader
Roll:
Flex

リーダーシップを発揮し、試合中の戦略立案を担当するIGLとしても活躍。彼のエイムは非常に正確で、特に置きエイムにおいて強さを発揮している。さらに、さまざまなエージェントを使いこなすフレキシブルなプレイヤーとしても評価されている。

Neru

Position:
Member
Roll:
Main Initiator

優れたエイム力を持ち、撃ち負けることがほとんどない。どんなに強い相手にも怯むことなく果敢に挑み、圧倒的な火力で試合を支配するスタイルが特徴。チームにとって不可欠な存在であり、試合の流れを大きく変える力を持っている。

shinchan

Position:
Member
Roll:
Main Duelist

圧倒的な攻撃力を誇るプレイヤー。どんなに固い防衛にも果敢に突っ込み、チームの先陣を切って敵陣を崩壊させる。エントリー能力に優れ、攻撃の起点となる存在。Shinchanの突破力があるからこそ、チームは強気の攻めを展開できる。

Rqin

Position:
Member
Roll:
Main Controller

チームの要となる存在。徹底的な努力を重ね、確実に実力を積み上げてきたプレイヤー。優れたサポート能力を持ち、試合の重要な局面で冷静にチームを支える。クラッチ力も高く、逆境の中でも的確な判断とプレイで勝利をもぎ取る。

Izayoi

Position:
Member
Roll:
Main Sentinel

チームの防衛の要を担うプレイヤー。敵の動きを細かく把握し、的確な情報を味方に提供することで、試合の流れをコントロールする。相手の傾向を冷静に分析し、最適なポジショニングでエリアを支配。
